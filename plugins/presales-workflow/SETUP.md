Complete the steps below after installing the plugin.

## External Dependencies

### tavily `[mcp_server]` *(moderate setup)*

**Install:**
```sh
Get a free API key at https://tavily.com, then replace YOUR_TAVILY_API_KEY_HERE in settings.json
```

**Docs:** https://github.com/tavily-ai/tavily-mcp

**Notes:** Free tier: 1,000 searches/month. Used for structured web research alongside mgrep --web.
